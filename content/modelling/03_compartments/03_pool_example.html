3.2. Пример – Пожар пролива#

Введение#

Пожары пролива интересны для изучения развития пожара, потому что с ними относительно легко обращаться. Нужен огнеупорный поддон, например, сделанный из стальных листов, и налейте в него горючую жидкость. Вскоре после воспламенения оно достигает квазистационарного состояния из-за испарения жидкости. Одним из недостатков пожара пролива является ограниченный запас топлива. Это одна из причин, по которой газовые горелки часто используются в экспериментах с огнем. Это, а также хорошая воспроизводимость, связанная с газовыми горелками в качестве источника воспламенения. Также в FDS большинство моделей пожара в бассейне моделируются в виде газовых горелок (см. Раздел “11.4 Простые модели пиролиза” в Руководстве пользователя FDS).

Этот пример иллюстрирует настройку и анализ минималистичного моделирования FDS. Начиная с простого пожара пролива с фиксированной скоростью тепловыделения, модель расширяется за счет различных параметров. Затем результаты проверяются путем сравнения с простыми аналитическими методами.

Настройка#

Сценарий пожара пролива рассматривается, как пример моделирования FDS. Площадь поверхности бассейна равна \(\mf 1~m^2\) и он находится \(\mf z = 0\). Скорость выделения тепла при пожаре равна \(\mf 10~MW\). Боковые и верхние границы области моделирования закрыты, а расширение вычислительной области ограничено .

\[\mf [-2.5~m, 2.5~m] \times [-2.5~m, 2.5~m] \times [0~m, 10~m]\]

с разрешением сетки \(\mf 0.2~m\).

Входной файл FDS#

Параметры во входном файле FDS задаются с помощью записей в формате списка имен. Каждая запись в списке имен начинается с символа амперсанда “&”, за которым следует название группы списков имен, затем список входных параметров, разделенных запятыми, и, наконец, косая черта “/”.

Tip

В таблице 21.3 Руководства пользователя FDS содержит краткое описание часто используемых записей.

Простой входной файл с комментариями, который может послужить отправной точкой для следующего упражнения, выглядит следующим образом:

"--------------------------------------------------------------------------------------------"
"Выходные файлы будут помечены тегами 'pool_fire', 'TITLE' это просто описание"
"--------------------------------------------------------------------------------------------"
&HEAD CHID = 'pool_fire', TITLE = 'Example of a pool fire'  /

"--------------------------------------------------------------------------------------------"
"Сетка с количеством ячеек 25/25/50 по осям X/Y/Z которая растянута X/Y/Z = -2.5/-2.5/0 to 2.5/2.5/10 m"
"--------------------------------------------------------------------------------------------"
&MESH IJK = 25,25,50, XB = -2.5,2.5, -2.5,2.5, 0.0,10 /

"--------------------------------------------------------------------------------------------"
"Время окончания моделирования - 30 с"
"--------------------------------------------------------------------------------------------"
&TIME T_END = 30.0 / End of simulation time is 30 s

"--------------------------------------------------------------------------------------------"
"Указание топлива FDS по умолчанию 'METHANE' для реакции"
"--------------------------------------------------------------------------------------------"
&REAC FUEL = 'METHANE' /

"--------------------------------------------------------------------------------------------"
"Поверхность «BURNER» со скоростью выделения тепла 1000 кВт/м^2"
"--------------------------------------------------------------------------------------------"
&SURF ID = 'BURNER', HRRPUA = 1000., COLOR = 'RASPBERRY' / Surface with ID /

"--------------------------------------------------------------------------------------------"
"Вент в плоскости X-Y при Z = 0, который простирается от X/Y = -0,5/-0,5 до 0,5/0,5"
"Поверхность 'BURNER' назначена для вентиляционного отверстия"
"--------------------------------------------------------------------------------------------"
&VENT XB = -0.5,0.5, -0.5,0.5, 0.0,0.0, SURF_ID='BURNER'/

"--------------------------------------------------------------------------------------------"
"Файл среза в плоскости Y-Z при X = 0 с величиной 'температура'"
"CELL_CENTERED предотвращает интерполяцию результатов FDS по углам ячеек."
"--------------------------------------------------------------------------------------------"
&SLCF PBX = 0, QUANTITY = 'TEMPERATURE', CELL_CENTERED = .TRUE. /

"--------------------------------------------------------------------------------------------"
"FDS не следит за этой линией"
"--------------------------------------------------------------------------------------------"
&TAIL /

Дополнительная задача: Моделирование чисто плавучего диффузионного пламени#

В этом задании мы предлагаем в качестве дополнительного упражнения моделирование диффузионного пламени на основе экспериментальной установки, описанной в отчете: Чисто плавучее диффузионное пламя: некоторые экспериментальные результаты. Чисто буйное диффузионное пламя можно определить как пламя, возникающее над горючими газами, покидающими поверхность твердого или жидкого топлива с незначительной скоростью. В таких случаях считается, что в возгорании преобладает плавучесть. Это общая характеристика пожаров, которые свободно развиваются над бассейном с жидким топливом или над твердым полимерным материалом.

Основная цель экспериментов, описанных в отчете, заключается в измерении температуры и величины скорости в ближнем поле пламени с помощью зондов и термопар. Горелка на природном газе впрыскивает топливо с регулируемой скоростью, чтобы получить желаемое тепловыделение. Задание II - Формулы шлейфа#

Целью данной задачи является вычисление температуры факела с помощью аналитических методов для последующей проверки результатов численного моделирования. Для получения более подробной информации о теории и применении формул шлейфа, пожалуйста, обратитесь к литературе, например, к B. Karlsson и J. Quintiere [KQ99].
